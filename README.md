## 🚀 Overview

BeepBite is a comprehensive restaurant order management system that provides real-time WhatsApp notifications, order tracking, analytics, and team management tools. Built for restaurants that want to streamline their operations and never miss an order during peak hours.

## ✨ Key Features

- **📱 WhatsApp Integration** - Instant order notifications via WhatsApp
- **⏰ Real-time Order Tracking** - Track orders from pending to ready
- **📊 Analytics Dashboard** - Order patterns, peak hours, and performance metrics
- **👥 Team Management** - Role-based access for owners, managers, and staff
- **⭐ Review System** - Collect and manage customer feedback
- **🔔 Smart Notifications** - Customizable alerts for different order statuses
- **📈 Reporting** - Comprehensive business insights and reports

## 🏃‍♂️ Quick Start

### Prerequisites

- Node.js 18+ 
- npm or yarn
- Modern web browser

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/beepbite-mono.git
cd beepbite-mono

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Edit .env.local with your configuration

# Start development server
npm run dev
```

Visit `http://localhost:5173` to see the application.

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS
- **UI Components**: Radix UI, shadcn/ui
- **Backend**: Supabase, Firebase
- **Notifications**: WhatsApp Business API
- **Charts**: Recharts
- **Animation**: Framer Motion
